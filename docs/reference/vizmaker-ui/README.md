# 실물 VizMaker UI 스크린샷 (2026-07-05, 사용자 제공 — UI 재현의 절대 기준)

12장. 명세 문서와 다르면 이 스크린샷이 우선.

핵심 관찰 (KakaoTalk_Photo...10-51 기준):
- 노드 = **큰 이미지 카드** (결과 썸네일이 카드 본체). 카드 아래 굵은 제목(Source / Nano Banana Pro / Kling o1) + 프롬프트 요약 텍스트
- 연결 = 카드 사이 작은 포트 점-선 (가늘고 미니멀), 캔버스 배경은 완전 무광 검정, 그리드 없음
- 좌측 레일: Projects / History / Org / Tutorial ... Account / Contact / Settings (아이콘+라벨, 상하 분리)
- 우측 패널: [Enlarge] 버튼 / Preview·Compare·Draw 탭 / Image Editor 섹션 / Engine 드롭다운(Nano Banana 2 + Precise·Quality·Fast 칩) / Priority(Standard, Relaxed 설명 툴팁) / Resolution / Aspect ratio / Prompt Presets·My Presets 그리드
- 하단: 프롬프트 입력 한 줄 + 새로고침 아이콘 + [Make ▶] 버튼
- 영상 노드는 카드에 ▶ 재생 버튼 오버레이
- 창 제목: "VizMaker - <프로젝트명>.vm"
